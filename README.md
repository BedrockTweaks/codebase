## Codebase

The source code from [https://bedrocktweaks.net](https://bedrocktweaks.net)

### Files Repository

The source files for resource packs, addons, and crafting tweaks used in this project are hosted in a separate repository: **[BedrockTweaks/Files](https://github.com/BedrockTweaks/Files)**

## Monorepo Structure

```
├── apps/
│   ├── web/          # Main React web application
│   └── api/          # Backend API service (closed source)
├── packages/
│   └── config/       # Shared configuration
├── turbo.json        # Turborepo pipeline configuration
└── pnpm-workspace.yaml
```
